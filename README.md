# Home SOC Lab

## Objective
Build a Security Operations Center lab using:
- Kali Linux
- Windows 10
- Wazuh SIEM

## Lab Architecture
Kali (Attacker) → Windows (Victim) → Wazuh (Monitoring)

## Skills Demonstrated
- Network scanning
- Vulnerability assessment
- Log analysis
- Incident response
